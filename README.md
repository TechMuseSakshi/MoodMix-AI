# MoodMix AI

MoodMix AI is a web-based music mood analyzer developed using Python, Flask, HTML, CSS, and Librosa.

## Features

* User Login and Signup
* Audio File Upload
* Music Mood Detection
* Tempo and Energy Analysis
* Interactive User Interface

## Technologies Used

* Python
* Flask
* HTML
* CSS
* Librosa
* NumPy

## Project Status

The project is currently under development. Future enhancements include advanced mood prediction, personalized music recommendations, and cloud deployment.
